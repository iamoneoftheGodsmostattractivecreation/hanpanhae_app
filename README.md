# hanpanhae 🎮

A real-time multiplayer party game app built with Flutter and Firebase.

This is my first real app development project.
I started this project to learn Flutter, Firebase, and real-time synchronization by building a multiplayer mini-game application from scratch.

---

## Features

### Mini Games
- Tap Game
- Rhythm Game
- Wrong Word Game
- Green Tile Reflex Game
- Picture Guess Game

### Multiplayer System
- Real-time Room Creation & Join
- Nickname System
- Host Management
- Ready / Cancel System
- Real-time Chat
- Real-time Player List Synchronization

### Game Flow Synchronization
- Game Selection Synchronization
- Punishment Selection Synchronization
- Time Selection Synchronization
- Automatic Game Start
- Room State Synchronization

### Result System
- Firebase Result Storage
- Multiplayer Score Collection
- Ranking System
